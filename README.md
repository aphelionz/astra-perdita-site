# astraperdita.com

Static link-tree landing page for *Astra Perdita (Four Piano Vignettes for Those Lost Exploring the Stars)* by Mark Robert Henderson.

Single static `index.html`, no build step. Served via GitHub Pages at the custom domain `astraperdita.com` (see `CNAME`).

## Edit

Edit `index.html` and push to `main`. GitHub Pages redeploys automatically.

## Files

- `index.html` — the page
- `cover.jpg` — album artwork (Scott Henderson)
- `CNAME` — custom domain for GitHub Pages
